Configuration générale

Les paramètres de base qui s'appliquent à l'ensemble de votre gabarit.

1.
Titre du gabarit
Nom visible dans le menu Mes gabarits lors de la génération d'une note.
2.
Langue
Langue de rédaction de vos notes : Français ou Anglais.
3.
Visibilité
·Privé : visible uniquement par vous.
·Public : partagé avec la communauté Plume IA.
4.
Directives générales
Instructions globales qui s'appliquent à tout le gabarit. Idéal pour définir le ton, la terminologie préférée ou des règles transversales.
Bon à savoir : Les directives générales s'appliquent globalement à toute la note. Pour des instructions propres à une seule section, utilisez le champ Contenu de cette section spécifique.
